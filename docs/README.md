## Project Overview
This project explores the classic **0/1 Knapsack Problem**, where the goal is to select a subset of experiments that maximizes total rating without exceeding a weight limit of 700 kg.

I implemented and compared multiple algorithmic strategies:
- Three Greedy approaches
- Brute Force (Exhaustive Search)
- Dynamic Programming (optimal solution)
